FX recap:
- XXX/YYY
- X → Base Currency
- Y → Quote Currency / Counter Currency
- so X quoted in Y
- e.g USD/CHF → How many USD can get with one CHF
	- → How many Y it takes to buy one X
	- → The price of X in Y
	- → How much of the quote currency you need to exchange for 1 unit of the base currency
	- → <mark style="background: #FF5582A6;">Selling</mark> quote to <mark style="background: #BBFABBA6;">buy</mark> base
- Buying X Using Y (selling Y)
- Buying USD against CHF
