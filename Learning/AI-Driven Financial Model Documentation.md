## Introduction
- This project **investigates** a:
	- forward‑looking,
	- AI‑driven approach
- To **transforming** how quantitative **documentation** is created
	- Within the **Financial Markets sector**.
---
- Our consultants often need to **generate** **pricing replication** and **detailed documentation** as part of
	- 1- product development
	- 2- system upgrades
	- 3- project delivery
- For tasks that are traditionally
	- manual,
	- time‑intensive,
	- prone to human error &
	- challenging to update as inputs evolve.
---
<br>

> This project proposes an automated, intelligent workflow designed to deliver:
> 	- 1. accuracy,
> 	- 2. consistency,
> 	- 3. and scalability.

<br>

### Embedding AI

---
<br>

![[IMG-20260312103550963.png]]

<br>

By embedding AI directly into the documentation lifecycle, the framework will *automatically*
- A - **extract** pricing data from **multiple source systems**,
- B - **interpret** complex **product structures**, and
- C - **produce** transparent **Excel‑based pricing sheets**
	- alongside comprehensive technical documentation

- This includes 
	- methodologies,
	- assumptions,
	- limitations,
	- and validation considerations

> —generated seamlessly and intuitively.

<br>

> [!tip] The result is a future‑ready documentation process that strengthens transparency, reduces operational burden, and *enables faster, more reliable product development*.

<br>

## Objectives

<br>

![[IMG-20260312103925177.png]]


The project's objectives are:


| Item | Objective          | Description                                                                                                                                                                                                                  |
| ---- | ------------------ | ---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| *1*  | 📚 **Literature**  | Review and collate existing literature, highlighting the importance of quantitative documentation, current challenges in quant modelling, and recent technological innovations using AI                                      |
| *2*  | 🚧**Framework**    | Develop a framework capable of interacting with financial systems and utilizing natural language prompts to automate the generation of model documentation.                                                                  |
| *3*  | 🧩**Train**        | Train advanced AI models to interpret complex quantitative inputs and autonomously generate accurate outputs given the inputs and product                                                                                    |
| *4*  | 🔢**Metrics**      | The pricing model outputs should include the basic trade metrics such as the market value, interpolated rates, sensitivities and possibly even value at risk                                                                 |
| *5*  | ⬅️**Ingestion**    | Implement AI-powered ingestion of model inputs, including market data, curves, volatility surfaces, trade terms, and reference data, ensuring seamless data integration.                                                     |
| *6*  | 🧠**Intelligence** | Utilize AI algorithms to intelligently select relevant, sufficient, and reliable model inputs tailored to the specific product being priced. The models might depend on the client implementation which could be considered. |
| *7*  | ➡️**Extraction**   | Automate the creation of Excel pricing sheets using AI, ensuring transparency, auditability, and usability for coursework and presentations.                                                                                 |
| *8*  | 🎯**Output**       | Employ AI to generate comprehensive model documentation, detailing methodologies, calibration processes, data requirements, and validation procedures.                                                                       |
| *9*  | ✅**Validation**    | Assess the AI-enabled framework's performance and user interface, focusing on improvements in efficiency, accuracy, and user satisfaction compared to traditional approaches.                                                |


# Project Description



![[IMG-20260312110407531.png]]



<br>

The project will progress through the following stages:


| Number | Stage Name                         | Description                                                                                                                                                                                               |
| ------ | ---------------------------------- | --------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| *1*    | **Literature Review**              | Conduct a comprehensive review of existing research on AI driven quantitative documentation, the challenges of traditional quant model documentation, and recent advancements in AI powered automation.   |
| *2*    | **AI Framework Development<br>**   | Design and develop an AI powered framework that integrates AI with financial data sources, enabling automated extraction, interpretation, and synthesis of quantitative pricing data and product details. |
| *3*    | **Documentation Generation<br>**   | Implement AI workflows to autonomously generate Excel based pricing sheets and detailed model documentation, including methodologies, assumptions, calibration, and validation processes.                 |
| *4*    | **User Interface Development<br>** | Build an intuitive front-end interface that allows users to interact with the AI driven documentation system using natural language prompts, ensuring accessibility and ease of use.                      |
| *5*    | **Performance Evaluation<br>**     | Rigorously test the AI enabled framework for efficiency, accuracy, transparency, and user satisfaction, benchmarking against traditional documentation and pricing model workflows.                       |


# Deliverables
<br>

![[IMG-20260312105959167.png]]

<br>

The project's outcomes include:
<br>

## 1. Literature Review Report

A summary of current research on AI driven quant documentation, how AI is used in finance, and the limitations of traditional model documentation.

## 2. AI Driven Framework

A system that uses AI to automatically pull in, interpret, and combine pricing data and product details from different financial sources.

## 3. Automated Excel Pricing Sheets

AI generated Excel files that provide transparent and auditable pricing calculations for specific financial products.

## 4. Model Documentation

Automatically produce documentation that explains each pricing model, including methods, assumptions, data needs, calibration, and validation steps.

## 5. User Interface Prototype

A simple front end interface that lets users interact with the system using natural language, along with basic user guidance.

## 6. Performance Evaluation Report
An assessment of how well the AI system performs compared to traditional processes, focusing on efficiency, accuracy, transparency, and user experience.
